# 🪪 Bilal Tabibi — Personal Name Card

A minimal, elegant personal portfolio card built with **Flask** and **pure HTML/CSS**. Features an animated fullscreen background, contact info, social links, and a message form.

---

## ✨ Features

- Animated background with slow zoom effect
- Elegant card UI with gold accents
- Social links (GitHub, LinkedIn, Twitter)
- Contact info — email, phone, location
- Contact form (name, email, message)
- Responsive design
- Google Fonts — Cormorant Garamond & Raleway
- Font Awesome icons

## 🚀 Getting Started

**1. Clone the repository**
```bash
git clone https://github.com/bilal-tabibi/bilal-tabibi.git
cd bilal-tabibi
```

**2. Create and activate a virtual environment**
```bash
python3 -m venv myenv
source myenv/bin/activate
```

**3. Install Flask**
```bash
pip install flask
```

**4. Run the app**
```bash
python3 app.py
```

**5. Open in browser**
```
http://127.0.0.1:5000
```
